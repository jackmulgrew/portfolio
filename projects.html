Rambler

April 2024

Website > Repository >

Rambler was created in my honours project in the last year of my university degree. The platform is built to allow travellers to save their itineraries for ease of access. The application stores the users important travel information as well as saving important documentation such as train tickets, visa information and hotel confirmations.

The application was created using the MERN stack. React was used to create the front end which the user interacts with. The back end was created using Express.js and Node.js which run the server side and connect the users interactions to the cloud database stored on MongoDB Atlas.
